# 🧮 Calculator + ✂️ Rock-Paper-Scissors

A versatile command-line tool that combines mathematical operations with a classic hand game.

## 🚀 Features
- **Basic Calculator**: Addition, subtraction, multiplication, division.
- **Mini-Game**: Play "Rock, Paper, Scissors" against the computer.
- **Simple UI**: Easy-to-use text-based interface.

## 🛠️ How to use
1. Start the program.
2. Choose between "Calculator" or "Game".
3. Follow the on-screen instructions.
